# 🐟 工作摸鱼小游戏 设计文档

## 游戏概述
- 类型：休闲/点击/解压
- 核心玩法：在"假装工作"的外表下，完成各种无聊但解压的食物加工小任务
- 目标用户：上班族、学生、需要解压的人
- 游戏名称：**摸鱼工坊**（Slacking Factory）

## 核心创意
玩家扮演一个"摸鱼大师"，在工位上偷偷进行各种食物加工。每个任务都是重复性的、令人上瘾的小操作。任务随机切换，完成越多分数越高。

## 技术方案
- 渲染：Canvas + HTML/CSS 混合（Canvas 绘制食物，CSS 做 UI）
- 交互：点击/触摸（手机端）+ 鼠标（PC端）
- 动画：Canvas requestAnimationFrame + CSS transitions
- 单文件 HTML 实现

## 美术风格
- 风格：可爱卡通 + 扁平设计
- 视角：正面视角（Front view）
- 光照：柔和明亮
- 配色：暖色调（米白色背景 + 彩色食物）
- 整体氛围：轻松、治愈、有趣

## 游戏模式
**无尽模式：**
- 随机分配任务，每个任务有时间限制
- 完成任务获得分数和金币
- 连续完成任务获得 Combo 加成
- 超时或失败扣减生命值
- 3 条命，命没了 Game Over

## 核心任务（Mini-games）

### 任务 1：给苏打饼干扎孔 🍪
- **操作**：点击/触摸饼干表面扎孔
- **目标**：在指定区域扎出 N 个均匀分布的孔
- **判定**：孔的数量、位置均匀度
- **视觉反馈**：
  - 点击时：饼干表面出现小圆孔 + 轻微下凹效果
  - 扎孔瞬间：小面粉粒子飞溅
  - 完成时：饼干翻转 ✓ 动画

### 任务 2：给曲奇饼干点巧克力豆 🍫
- **操作**：点击/触摸曲奇表面放置巧克力豆
- **目标**：在合适位置放置 N 个巧克力豆
- **判定**：巧克力豆不能重叠、不能超出饼干边缘
- **视觉反馈**：
  - 点击时：巧克力豆从上方掉落嵌入面团
  - 放置瞬间：轻微的挤压变形
  - 完成时：曲奇闪金光 ✓ 动画

### 任务 3：给草莓去籽 🍓
- **操作**：点击/触摸草莓表面的黄色小籽
- **目标**：去除草莓表面的所有小籽
- **判定**：去除的籽数量
- **视觉反馈**：
  - 点击籽时：籽弹飞出去 + 小水珠飞溅
  - 去除后：该位置变成光滑的红色表面
  - 完成时：草莓闪亮 ✓ 动画

### 任务 4：给蛋糕裱花 🎂
- **操作**：沿着虚线轨迹滑动/拖拽
- **目标**：沿着预设路径挤出奶油花纹
- **判定**：路径覆盖率、偏离程度
- **视觉反馈**：
  - 滑动时：白色奶油线条跟随手指
  - 奶油有立体感（厚度渐变）
  - 完成时：撒花 ✓ 动画

### 任务 5：给面包片涂黄油 🧈
- **操作**：在面包片上来回滑动
- **目标**：用黄油覆盖面包表面
- **判定**：覆盖面积百分比
- **视觉反馈**：
  - 滑动时：黄油刀跟随手指，留下金黄色涂抹痕迹
  - 涂抹区域变亮变黄
  - 完成时：面包飞走 ✓ 动画

### 任务 6：剥橘子 🍊
- **操作**：从底部向上滑动剥皮
- **目标**：完整剥掉橘子皮
- **判定**：剥离面积百分比
- **视觉反馈**：
  - 滑动时：橘子皮向外翻卷
  - 露出里面的橙色果肉
  - 完成时：橘子瓣散开 ✓ 动画

## 交互细节

### 通用视觉反馈
- **点击/触摸**：触摸点出现涟漪效果（圆环扩散）
- **任务开始**：食物从屏幕外弹入，带弹性效果
- **任务完成**：食物缩小飞走 + ✓ 图标 + 得分浮字
- **任务失败**：食物抖动 + ✗ 图标 + 红色闪烁
- **Combo**：屏幕边缘发光，Combo数字放大弹跳

### 状态变化
- **时间条**：绿色 → 黄色（<50%）→ 红色（<20%）→ 闪烁
- **分数变化**：数字滚动 + 金色粒子
- **生命值变化**：心形图标弹跳 + 碎裂动画

## 评分机制
- 基础分：每个任务 100 分
- 速度加成：越快完成分数越高（最高 1.5x）
- 完美加成：完美完成额外 +50 分
- Combo 加成：连续完成 × 1.1, 1.2, 1.3...（最高 2.0x）
- 最终得分 = 基础分 × 速度加成 × 完美加成 × Combo加成
