# 🐟 摸鱼工坊 (Slacking Factory)

一款轻松解压的工作摸鱼小游戏！在假装工作的同时，完成各种无聊但令人上瘾的食物加工任务。

## 🎮 游戏玩法

在工位上偷偷进行各种食物加工，每个任务都是重复性的、令人上瘾的小操作：

- 🍪 **给苏打饼干扎孔** - 点击扎出均匀的小孔
- 🍫 **给曲奇饼干点巧克力豆** - 点击放置巧克力豆
- 🍓 **给草莓去籽** - 点击去除草莓表面的籽
- 🎂 **给蛋糕裱花** - 拖拽绘制奶油花纹
- 🧈 **给面包片涂黄油** - 滑动涂抹黄油
- 🍊 **剥橘子** - 向上滑动剥掉橘子皮

## ✨ 游戏特色

- 🎯 **6种核心任务** - 点击和拖拽两种操作方式
- ⏱️ **时间挑战** - 每个任务30秒限时
- 💯 **评分系统** - 速度加成 + Combo连击
- 🎨 **精美视觉** - 粒子效果 + 流畅动画
- 🔊 **悦耳音效** - 程序化生成的音效
- 📱 **完美适配** - PC和移动端都流畅

## 🎯 游戏目标

- 在时间限制内完成任务
- 连续完成任务获得 Combo 加成
- 挑战更高分数
- 3条命用完游戏结束

## 🎮 操作说明

### PC端
- **点击类任务**：鼠标点击
- **拖拽类任务**：鼠标按住拖动

### 移动端
- **点击类任务**：手指点击
- **拖拽类任务**：手指滑动

## 📊 评分机制

```
最终得分 = 基础分(100) × 速度加成(1.0-1.5) × 完美加成(+50) × Combo加成(1.1-2.0)
```

- **速度加成**：越快完成分数越高
- **完美加成**：完美完成额外+50分
- **Combo加成**：连续完成任务，最高2倍
